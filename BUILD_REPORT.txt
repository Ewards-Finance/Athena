API ENDPOINTS SUMMARY
--------------------------------------------------------------------------------
POST   /api/auth/login                → Returns JWT
GET    /api/auth/me                   → Current user info
GET    /api/dashboard/stats           → Stat card data
GET    /api/employees                 → List all (Admin/Manager)
GET    /api/employees/:id             → Single employee profile
POST   /api/employees                 → Create employee (Admin)
PUT    /api/employees/:id             → Update profile
DELETE /api/employees/:id             → Deactivate (Admin)
GET    /api/leaves                    → List leaves
GET    /api/leaves/pending            → Pending approvals
POST   /api/leaves                    → Apply for leave
PATCH  /api/leaves/:id/approve        → Approve leave
PATCH  /api/leaves/:id/reject         → Reject leave
DELETE /api/leaves/:id                → Cancel leave
GET    /api/claims                    → List claims
POST   /api/claims                    → Submit claim
PATCH  /api/claims/:id/approve        → Approve claim
PATCH  /api/claims/:id/pay            → Mark as paid (Admin)
PATCH  /api/claims/:id/reject         → Reject claim
GET    /api/holidays                  → List holidays
POST   /api/holidays                  → Add holiday (Admin)
DELETE /api/holidays/:id              → Remove holiday (Admin)
GET    /api/announcements             → List announcements
POST   /api/announcements             → Post announcement (Admin)
DELETE /api/announcements/:id         → Deactivate announcement (Admin)
GET    /api/health                    → Health check
